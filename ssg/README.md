# `ssg`

A component-oriented static site generator.

Given this source code:

```
site/
└── index.rb
```

```rb
class Index < SSG::Page
  def render
    "<p>Hello World!</p>"
  end
end
```

This will be generated by `bundle exec ssg`:

```
_site/
└── index.html
```

```html
<p>Hello World!</p>
```
